# House Price Prediction

## Project Overview
This project predicts California house prices using Machine Learning. The California Housing dataset from Scikit-learn is used for training and testing the model.

## Dataset
- California Housing Dataset
- Source: Scikit-learn

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Machine Learning Models
- Linear Regression
- Random Forest Regressor

## Steps Performed
1. Loaded the dataset
2. Explored the data
3. Checked missing values
4. Split the data into training and testing sets
5. Trained a Linear Regression model
6. Trained a Random Forest Regressor
7. Compared model performance
8. Predicted house prices
9. Saved the trained model

## Results
The Random Forest Regressor achieved better prediction accuracy than Linear Regression.
